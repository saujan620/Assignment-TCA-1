Site Report

The experience of learning the basics of web development, coding and designing is an exciting journey which builds the basics foundation for creating attractive functional websites.This experience really helps me to develop my creativity and skills of learning new code and different property.Studying web development is not just learning about html and css property but also helps to develop new idea like solving problems, generating new techniques..Somehow,I find it's challenging but enjoying and satisying when websites run perfectly.

So, writing about the projects, I built a portfolio website using html and css. In this I have used different property with uniform header that includes logo and navigation menu across all pages.The webpages i.e. index.html has different html tags for headings, paragraphs, images and social media link.Similarly,In project.html file, I have used different grid layout containing project objects, photos and descriptions.Likewise, I have used a form with input fields for name, email messages, and a submit button with my contact information in contact.html file.Using CSS, I created a responsive webpages with simple grid layouts and media queries to adjust different screen sizes. I have used different color scheme in text and background in a simple way.Similarly,I have used different font-size and padding or gapping in different pages.I used hover effects on navigations links which provide attractions towards my pages.
